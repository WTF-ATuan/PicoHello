Description:

Loading Spinner Creator is component which creates unique loading spinner animations using
trigonometric functions with parameters setted in way, to make big changes for animation
appearance but still look cool and hypnotizing.
Feel free to change code a bit to animate it more, like animating "Small Dots" or animating
parameters variables to make it even more unique and cool to look at.
