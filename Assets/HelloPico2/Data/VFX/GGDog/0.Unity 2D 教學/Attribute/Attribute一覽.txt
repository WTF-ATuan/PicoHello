
-------------------------------------------------------------------------------------

隱藏變數

[HideInInspector]

讓公用變量不顯示在Inspector中，可以隱藏不需要手動更改的public變量。

[HideInInspector]
public int myVar = 1 ;

-------------------------------------------------------------------------------------

變數範圍

[Range(min,max)]

因為把變量限制在範圍內，所以會出現一個拉桿。可避免製作者設定超過容許範圍的數值

-------------------------------------------------------------------------------------

顯示變量(使private可顯示)

[SerializeField]

與HideInInspector相反，可以讓private變量顯示在Inspector中，讓製作者可以自由修改

-------------------------------------------------------------------------------------

標註變數

[Header("SomeText")]

給腳本一個標題，這會顯示在Inspector中，可以把變量隔開、註解

[Header("某個整數")] public int myInt = 0 ;
[Header("")] // 可以當作空行
public string myString = "" ;

-------------------------------------------------------------------------------------

標籤變數

Tooltip("SomeText")

給變量一個 "提示"，滑鼠指著他會出現文字


[Tooltip("你的名字")]
public string yourName = "你的名字" ;


-------------------------------------------------------------------------------------


文字框

[Multiline]

讓string字串變成多行形式，也就文字框

\n是跳行


[Multiline] public string yourName = "你的\n名字" ;


[Multiline(版面行數)]

[TextArea(版面最小行數，版面最大行數)] 如果沒打字，顯示框就是最小行數，超過最大行數就會變成拉條
